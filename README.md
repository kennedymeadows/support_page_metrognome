# Metrognome Metronome - Support Page

A beautiful, responsive support page for the Metrognome Metronome app, ready to deploy on GitHub Pages.

## Quick Start

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Your support page will be live at `https://yourusername.github.io/support_page_metrognome`

## Features

- ✨ Beautiful gradient design
- 📱 Fully responsive (mobile, tablet, desktop)
- ❓ Interactive FAQ section with collapsible answers
- 🔧 Comprehensive troubleshooting guide
- 💡 Tips & tricks section
- 📧 Contact information
- ⚡ Fast load - single HTML file, no dependencies
